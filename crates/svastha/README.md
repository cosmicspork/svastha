# svastha

Umbrella crate for [Svastha](https://github.com/cosmicspork/svastha) —
self-custodial, end-to-end-encrypted, local-first personal medical records. It
re-exports [`svastha-core`](https://crates.io/crates/svastha-core), the trust
contract (encryption envelope, event schema, relay protocol); the public surface
is identical to `svastha-core` under the bare `svastha` name.
