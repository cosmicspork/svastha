# svastha-node

The [Svastha](https://github.com/cosmicspork/svastha) node: a trusted processing
client that holds keys, syncs plaintext locally, and runs OCR / extraction /
de-identification / RAG by delegating inference to a user-supplied
OpenAI-compatible endpoint (it ships no models). A later release; currently a
skeleton.

> Pre-1.0 and unstable.
